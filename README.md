# WordPress Project

This project is built with **WordPress** and runs using **Docker**.

## Requirements

- Docker
- Docker Compose

## Getting Started

1. Open a terminal in the project root.
2. Start the containers:

```bash
docker compose up
```

3. Once the containers are running, open your browser and navigate to:

```
http://localhost
```

If this is the first time running the project, WordPress will guide you through the initial installation process.

## Stopping the Project

To stop the containers, press `Ctrl + C` in the terminal or run:

```bash
docker compose down
```
